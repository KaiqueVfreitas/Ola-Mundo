Principais Formataçoes

Negrito

Usando a tag B podemos deixar as frases em negrito(nao semantica)

Usando o strong é possivel adicionar negrito tambem so que desta vez com semantica


Italico/Enfase

A tag I serve para adicionar italico (Sem samantica)

Usando a tag EM podemos adicionar italico tambem mas desta vez com semantica


tag mark

com a tag mark da para grifar textos


Textos maiores e menores com as tags

Com a tag small é possivel ter letras pequenas

Com a tag big é possivel deixar os textos um pouco menores


Textos Inseridos e Deletados

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois


Textos Sobrescrito e Subscrito

Com a tag sob é possivel adicionar um caractere em CIMA de outro x2- 0

Com a tag sUb é possivel adicionar um caractere em BAIXO de outro H2O
